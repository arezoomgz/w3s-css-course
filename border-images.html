CSS Border Images

With the CSS border-image property, you can set an image to be used as the border around an element. Here, the middle sections of the image are repeated to create the border:

border-image: url(border.png) 30 round;

Here is the original image:

Here, the middle sections of the image are stretched to create the border:

border-image: url(border.png) 30 stretch;

Different slice values completely changes the look of the border.

border-image: url(images/bord.png) 19% round;


border-image: url(images/bord.png) 50% round;
